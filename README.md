# MusicMood

####
MusicMood is a online community for people to connect via music and their moods. Users simply choose a mood and upload a video that matches their mood.
---
#### Live site: http://music-mood-app.herokuapp.com/

#### Built With
* Ruby on Rails
* Javascript
* SQL
* React.js
* HTML5, CSS3
* Bootstrap
